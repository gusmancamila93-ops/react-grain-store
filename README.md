# Grain Store

Sistema administrativo para la gestión de una tienda de granos desarrollado como proyecto académico ADSO SENA utilizando React, Vite, Tailwind CSS y Shadcn/UI.

## Descripción

Grain Store permite administrar las operaciones principales de una tienda de granos mediante una interfaz moderna, intuitiva y responsive.

El sistema incluye autenticación por roles y módulos para la gestión de productos, clientes, ventas, reportes, egresos y usuarios.

## Características

- Autenticación simulada
- Gestión de roles
  - Administrador
  - Vendedor
  - Contador
- Dashboard interactivo
- Gestión de productos
- Gestión de clientes
- Gestión de ventas
- Gestión de egresos
- Reportes con datos simulados
- Configuración del sistema
- Administración de usuarios
- Interfaz responsive
- Modo oscuro

## Tecnologías utilizadas

## Instalación

Clonar el repositorio:

```bash
git clone <url-del-repositorio>
```

Ingresar al proyecto:

```bash
cd react_js_Grain_estore
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```

Generar compilación de producción:

```bash
npm run build
```
